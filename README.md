# Mini.Chess.Player

A simplistic implementation of a Minichess player using Kotlin.

## Overview

This repository contains the basic framework for the client with AlphaBeta pruning in Kotlin.  The original basis for this code was adapted from [Minichess](https://github.com/BartMassey/IMCS).
